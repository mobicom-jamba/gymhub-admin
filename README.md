# GymHub Admin Dashboard

GymHub Admin is a gym management admin dashboard built on **Next.js and Tailwind CSS** with **Supabase** as the backend.

![GymHub Admin Dashboard](./banner.png)

## Overview

Built on:

* Next.js 16.x
* React 19
* TypeScript
* Tailwind CSS V4
* Supabase

## Installation

### Prerequisites

* Node.js 18.x or later (recommended to use Node.js 20.x or later)

### Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/mobicom-jamba/gymhub-admin.git
   ```

2. Install dependencies:

   ```bash
   npm install
   # or
   yarn install
   ```

3. Copy `.env.local.example` to `.env.local` and fill in your Supabase credentials.

4. Start the development server:

   ```bash
   npm run dev
   # or
   yarn dev
   ```

## Features

* Gym management dashboard
* User management
* Booking & scheduling
* Visit tracking
* Data visualization with charts
* Authentication via Supabase
* Dark mode support
* Responsive design
